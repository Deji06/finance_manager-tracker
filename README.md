--Personal Finance Tracker

Managing money in a fast-paced economy like Nigeria’s is tough—tracking data bundles, fuel costs, unexpected owambe expenses, and salary extras can get messy fast. NairaSmart was designed to solve that by providing a high-performance, visually clean way to master your cash flow.

The Finance Tracker App was built to solve this problem by giving users a clean, structured, and intuitive interface to track their daily expenses. It helps users stay financially organized by recording expenses, categorizing them, and presenting them in an easy-to-understand format.

This project showcases my ability to design and build a full-stack application, implement secure authentication, structure scalable APIs, and build a responsive modern UI.

🔗 Live App Link:
https://finance-manager-tracker.vercel.app/

--Key Features
 ** Expense Tracking – Easily record daily expenses
 ** Category Organization – Group expenses by categories such as food, transport, rent, etc
 ** Date-based Grouping – Automatically groups expenses by day for easier viewing
 ** Update & Delete Expenses – Full CRUD functionality
 ** User Authentication – Secure login and registration
 ** Responsive UI – Works across mobile, tablet, and desktop
 ** Fast Data Fetching – Powered by TanStack Query for optimized API interactions
 
--Tech Stack
-Frontend
** React + TypeScript
** Vite
** TailwindCSS
** React Hook Form + Zod (form validation)
** TanStack Query (data fetching & caching)

-Backend
** Express.js
** Prisma ORM
** Database
** PostgreSQL
** Hosted on Supabase

-Deployment
** Frontend: Vercel
** Backend: Render

Together these technologies create a modern, scalable full-stack architecture capable of handling real-time financial tracking efficiently.

--Project Setup
clone repository -- git clone https://github.com/Deji06/finance_manager-tracker.git
install dependencies -- npm install
run development server -- npm run dev
